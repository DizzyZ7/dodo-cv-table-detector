# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-01

Initial public release.

### Added

- single-table occupancy detection from video
- manual and interactive ROI selection
- background initialization from early frames
- adaptive background update during `empty` state
- annotated output video export
- CSV export for events and state segments
- JSON summary export
- text report export
- repository documentation and MIT license

### Notes

- designed as a lightweight heuristic baseline
- intended for experimentation and small-scale CV prototyping
